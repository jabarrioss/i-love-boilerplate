# assets/maps

Tile maps, JSON level data, custom level formats.

Suggested libraries for tile-based games:
- [STI](https://github.com/karai17/Simple-Tiled-Implementation) — Tiled .tmx loader
- [anim8](https://github.com/kikito/anim8) — sprite sheet animations
- [bump](https://github.com/kikito/bump.lua) — collision detection
