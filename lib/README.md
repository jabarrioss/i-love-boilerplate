# lib

Drop third-party Lua libraries here. Anything inside this directory
is on `package.path` automatically.

Recommended:
- [bump](https://github.com/kikito/bump.lua)        — collision
- [anim8](https://github.com/kikito/anim8)         — sprite animation
- [STI](https://github.com/karai17/Simple-Tiled-Implementation) — Tiled maps
- [hump](https://github.com/vrld/hump)             — game-state + timer + vector
- [flux](https://github.com/rxi/flux)              — tweening
- [Classic](https://github.com/rxi/classic)        — OOP (alternative to core/Class.lua)
- [inspect](https://github.com/kikito/inspect.lua) — table pretty-printing
- [lume](https://github.com/rxi/lume)              — utility belt
